# StudyAI - Premium Purple Theme Design

## Design System Overview

StudyAI now features a sophisticated purple-based color system inspired by the reference design at `studyai.my/landing`.

### Color Palette

**Primary Colors:**
- **Deep Background**: `#0f0f23` - Dark navy-purple for elegant, professional appearance
- **Card/Surface**: `#1a1a3f` - Slightly lighter purple for layered depth
- **Accent (Purple)**: `#a78bfa` - Soft, sophisticated purple for CTAs and highlights

**Text & UI:**
- **Foreground**: `#f5f7fa` - Clean, readable off-white
- **Muted**: `#6b7280` - Subtle gray for secondary content
- **Border**: `#2d2454` - Purple-tinted borders for cohesion

**Charts:**
- Purple, Blue, Green, Yellow, Red for data visualization

### Typography

- **Sans-serif**: Poppins (300-900) - Modern, professional body text
- **Serif**: Cormorant Garamond - Elegant headings
- **Monospace**: Fira Code - Technical content

### Key Design Elements

1. **Gradient Buttons**: Purple-to-purple accent gradients on CTAs
   - "Get Started" buttons use `from-purple-500 to-accent`
   - Creates cohesive, premium look

2. **Gradient Text**: Hero headline "Not Harder" uses purple gradient
   - `from-purple-400 via-accent to-purple-400` for visual impact

3. **Glass Morphism**: Navigation uses `backdrop-blur-2xl` with transparency
   - Creates layered, modern aesthetic

4. **Animated Particles**: Background gradient orbs add depth without clutter
   - `bg-accent/5` and `bg-accent/3` blurred circles

5. **Smooth Animations**: 
   - slideUp, slideDown, slideIn for entrances
   - scaleIn for interactive elements
   - float for subtle motion

### Component Updates

**Landing Page:**
- Hero: "Study Smarter, Not Harder" with purple gradient
- CTA Buttons: Purple-to-accent gradient
- Feature Cards: Glass morphism style with hover effects
- Footer: Subtle background with clean navigation

**Dashboard:**
- Welcome header with purple tint
- Stat cards with colored gradients
- Tool cards with glass morphism and hover effects
- Premium icon styling

### Brand Consistency

All buttons, accents, and interactive elements now use the cohesive purple color system, creating a unified, professional brand experience that matches premium SaaS platforms.
